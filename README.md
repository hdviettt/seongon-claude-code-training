# Claude Code Training

Materials cho các buổi đào tạo Claude Code do Việt thiết kế và dẫn dắt.

Đối tượng chính: **non-tech** — marketer, quản lý, người không lập trình — học cách dùng Claude Code để ship sản phẩm thật, không phải học code.

## Triết lý

- Dạy **đọc bản đồ**, không dạy lập trình.
- Mỗi buổi học để lại 1 artifact ship-được (tool nội bộ, landing page, automation).
- Vocabulary tối thiểu, danger zones rõ ràng, mental model trước syntax.

## Cấu trúc 1 session

Mỗi folder session chứa:
- `README.md` — overview + outline
- `slides/` — slide deck source (Keynote/Google Slides/markdown)
- `handouts/` — cheatsheet, take-home
- `demos/` — demo project hoặc video link
- `exercises/` — bài tập hands-on

## Cách dùng repo này

- Là materials gốc — phiên bản trình bày (slide, video) build từ đây.
- Mỗi session có thể fork ra branch riêng khi customize cho 1 nhóm cụ thể (ví dụ session cho SEONGON khác cho khoá public).
