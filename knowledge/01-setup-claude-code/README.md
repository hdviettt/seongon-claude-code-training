# 01 — Setup Claude Code

## Khi nào học viên cần folder này

- Mới biết đến Claude Code, chưa cài.
- Cài xong nhưng không vào được.
- Mở Claude Code nhưng không biết gõ gì đầu tiên.
- Cần ôn lại các phím tắt, lệnh cơ bản (`@`, `/`, `!`, `esc+esc`).

## Tóm tắt

Claude Code là **trợ lý AI lập trình dạng dòng lệnh (CLI)** do Anthropic phát triển. Khác chatbot bình thường ở chỗ nó:
- Đọc và sửa được file trên máy bạn.
- Chạy được lệnh terminal.
- Kết nối được với hệ thống ngoài (API, MCP).
- Lưu context trong file thay vì chỉ trong đoạn chat.

Để bắt đầu, bạn cần:
1. Tài khoản Claude (Anthropic).
2. Git.
3. Terminal.
4. Claude Code.
5. Bun và Python (cho các tác vụ phát triển).

## Files trong folder

- `windows.md` — Cài Claude Code trên Windows từng bước.
- `mac.md` — Cài Claude Code trên Mac từng bước.
- `giao-dien-co-ban.md` — Sau khi cài xong, làm quen giao diện: lệnh, phím tắt, plan mode, approval.
- `troubleshooting.md` — Các lỗi thường gặp khi cài.
