# 05 — Tự động hoá Claude Code

## Khi nào học viên cần folder này

- Muốn dùng Claude Code khi không ở máy tính (trên điện thoại).
- Đã quen 4 buổi trước, muốn refine workflow để dùng hiệu quả nhất.
- Lo về cost — muốn biết alternatives.
- Đang phân vân giữa Claude Code và các tool khác.

## Tóm tắt

3 chủ đề:

### 1. Remote control
Dùng Claude Code trên điện thoại / iPad qua app Claude.

### 2. Best practice
Mindset đúng để dùng Claude Code hiệu quả: "việc gì Claude Code cũng làm được" → lãng phí tối thiểu, chất lượng tối đa.

### 3. Alternatives
Khi Claude Code quá đắt cho use case của bạn — các option khác: Codex, Antigravity, Cursor, OpenCode.

## Files trong folder

1. [`remote-control.md`](remote-control.md) — Dùng Claude Code trên điện thoại.
2. [`best-practice.md`](best-practice.md) — Mindset và workflow chuẩn.
3. [`alternatives.md`](alternatives.md) — Các tool thay thế Claude Code.

## Prerequisites

- Đã hoàn thành 4 folder kiến thức trước (setup → tech stack → mở rộng → agents/memory/hooks).
