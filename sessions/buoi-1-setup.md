# Buổi 1 — Setup Claude Code

## Mục tiêu buổi học

- Setup được Claude Code trên máy.
- Hiểu sự khác biệt của Claude Code so với chatbot phổ thông.
- Nắm được giao diện cơ bản.
- Tạo được 1 landing page.

## Đối tượng

Marketer / quản lý / non-tech, lần đầu tiếp xúc Claude Code.

## Thời lượng

3 giờ (gồm 10 phút giải lao).

## Outline

### Phần 1 — Cài đặt Claude Code (40 phút)

**Mục tiêu:** Mọi học viên có Claude Code chạy được trên máy.

Tham chiếu kiến thức:
- [`knowledge/01-setup-claude-code/windows.md`](../knowledge/01-setup-claude-code/windows.md)
- [`knowledge/01-setup-claude-code/mac.md`](../knowledge/01-setup-claude-code/mac.md)
- [`knowledge/01-setup-claude-code/troubleshooting.md`](../knowledge/01-setup-claude-code/troubleshooting.md)

Activities:
- Walkthrough cài Git, Terminal, Claude Code, Bun, Python.
- Trợ giảng hỗ trợ riêng từng học viên không cài được.

### Challenge giữa buổi (10 phút) — Tạo landing page

**Mục tiêu:** Trải nghiệm "tay không build landing page trong 10 phút" trước khi học lý thuyết.

Đề:
- Tự chọn 1 trong 2:
  - Clone landing page của 1 thương hiệu bất kỳ.
  - Tạo 1 landing page giới thiệu bản thân (có hình ảnh).
- **Không hỏi trợ giảng** — chỉ hỏi Claude Code.
- Cap màn hình hoặc quay lại, share lên nhóm cuối buổi.

### Phần 2 — Claude Code là gì? (30 phút)

**Mục tiêu:** Hiểu Claude Code khác chatbot bình thường ở đâu, biết khi nào dùng.

Talking points:
- AI vs Automation vs AI-in-Automation (n8n / Make / Zapier).
- Claude Code = AI mang sức mạnh lập trình.
- 2 môi trường Claude Code can thiệp: máy tính cá nhân + hệ sinh thái ngoài.
- Loop "Lấy thông tin → Lên kế hoạch → Hành động".

Demo: showcase 6 use case:
1. Viết bài content hàng loạt theo checklist.
2. Đăng bài WordPress hàng loạt.
3. Tạo landing page nhanh.
4. AI agent làm ảnh.
5. Phân tích AI Overviews.
6. Research & viết report.

### Phần 3 — Tại sao có khoá học này? (20 phút)

**Mục tiêu:** Tạo niềm tin "tôi cũng làm được", set kỳ vọng dài hạn.

Talking points:
- Claude Code thay đổi cách con người làm việc.
- Việc tốn hàng tháng → còn hàng giờ.
- Việc tưởng chừng không thể → thành có thể.
- Học CC = "kiến trúc cuộc sống của mình", không chỉ tăng năng suất.

### Phần 4 — Làm quen giao diện (50 phút)

**Mục tiêu:** Học viên dùng được các phím tắt, lệnh cơ bản, plan mode.

Tham chiếu kiến thức:
- [`knowledge/01-setup-claude-code/giao-dien-co-ban.md`](../knowledge/01-setup-claude-code/giao-dien-co-ban.md)

Cover:
- `@` tag file.
- `/` commands và SKILLs.
- Phím tắt (mũi tên, Enter, Esc, Alt+V dán ảnh, Esc+Esc xoá prompt).
- Plan mode (Shift+Tab 2 lần hoặc `/plan`).
- Approval flow.
- Lựa chọn đơn / đa.
- `!` bash mode.
- Chia terminal.

### Phần 5 — Tổng kết (20 phút)

5 điểm khác biệt CC vs chatbot phổ thông:
1. Không lặp lại ý — context lưu trong file.
2. Xây hệ thống tích lũy theo thời gian.
3. Làm việc đa nhiệm (multi-agent).
4. Bạn là kiến trúc sư giải pháp, không phải người viết code.
5. Không cần biết code, nhưng phải hiểu cách giao việc.
